This is a list of VLC shortcuts tailored for my needs.

| Shortcut | Function |
|-----------------|----------------|
| E               | frame by frame |
| F               | fullscreen |
| V               | subtitles on/off |
| Ctrl+Up/Down    | volume up/down |
| Shift+Left/Right   | +/- 3 seconds |
| Alt+Left/Right     | +/- 10 seconds |
| Ctrl+Left/Right    | +/- 1 minute |
| []=                | 10% slower/10% faster/normal speed |
